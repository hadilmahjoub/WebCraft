About The Course

"50 Projects 50 Days" is a collection of web development projects created by following an online course that teaches the fundamentals of HTML, CSS, and JavaScript, offered by Brad Traversy and Florin Pop. This is a pure, practical and fun project based course. Each project focuses on a specific skill or concept, from creating responsive web pages to building dynamic user interfaces, every project is different and while we use do use some repeated practices, they are each meant to teach you something new. By the end, you will have a stronger foundation in JS as well as HTML/CSS and should be able to move on and build your own projects.

What You Should Know

A very basic understanding of HTML, CSS & JavaScript is all that you need to know before taking this course. Things like common HTML5 tags, basic styling, JS variables, conditionals, etc

What you will learn

  • Project based teaching to sharpen your HTML, CSS & JavaScript skills
  • Perfect for beginners looking for things to build unique projects in a short timeframe
  • Modern styling with flexbox, layouts & UI's, CSS animations, custom properties, etc
  • Fundamental JS in real projects: triggers, events, array methods, Local storage, Promises & Async/Await, etc
  • DOM selection & manipulation, & more
  • JSON: HTTP requests, Fetching data & working with 3rd party APIs
  • Drag & Drop, Speech Recognition, Speech Synthesis, Canvas API & Animation
